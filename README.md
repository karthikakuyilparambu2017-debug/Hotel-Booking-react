The Hotel Room Booking System is a web-based application developed using React.js that allows users to browse available rooms, view room details, make reservations, and manage their booking history. The application provides a simple and user-friendly interface for booking hotel rooms online.
Users can view room information, check pricing details, book rooms, edit reservations, and cancel bookings. The system also maintains booking history for better reservation management.

Features
> Home Page
> View Available Rooms
> Room Details Page
> Room Booking Form
> Booking History
> Edit Booking Information
> Cancel Bookings
> Search Booking Records
> Local Storage Support
> Responsive Design

Technologies Used
> React.js
> JavaScript (ES6)
> HTML5
> CSS3
> React Router DOM
